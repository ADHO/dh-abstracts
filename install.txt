## dh-abstracts
install.txt

Two methods: 

1. Move the unarchived or unzipped directory into the Webapps directory of a Tomcat server. Make adjustments as needed to config files. 

2. Build XTF 3.1 (with mvn/ant) http://xtf.cdlib.org/documentation/deployment-guide/, and overlay the following folders from this package: brand, conf, css, icons, and style.
